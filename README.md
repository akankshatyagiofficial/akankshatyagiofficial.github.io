# Akanksha's Personal Website

This is my personal website built with Jekyll and hosted on GitHub Pages. The website showcases my work as a data analyst, including my projects and experience.

## Structure
- All content (Home, About, Projects) is now on a single page (`index.md`).
- The navigation bar at the top uses tabs that scroll to each section.
- There are no separate About or Projects pages anymore.

## Technologies Used
- Jekyll
- GitHub Pages
- HTML/CSS
- Markdown

## Local Development
To run this website locally:
1. Install Jekyll
2. Clone this repository
3. Run `jekyll serve`
4. Visit `http://localhost:4000`
